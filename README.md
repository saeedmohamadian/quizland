## How to Compile & Run:
1. Extract this ZIP file into a clean folder.
2. Open **Android Studio** (Hedgehog or higher recommended).
3. Click **File -> Open** (or Import Project) and select the extracted folder.
4. Let Android Studio sync the Gradle wrapper files (requires active internet connection).
5. **CRITICAL FOR PUSH NOTIFICATIONS**:
   - Go to your Firebase Console (https://console.firebase.google.com).
   - Create an Android App with package name `com.quizland.app`.
   - Download the generated `google-services.json` file.
   - Replace the empty template `app/google-services.json` file in this project with your downloaded one.
6. Click **Build -> Build Bundle(s) / APK(s) -> Build APK(s)** to compile your app.
7. Install the compiled `.apk` file on your phone or launch it on an emulator.

Enjoy your WebView container app!
